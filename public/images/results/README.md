## Bild-Anforderungen

- Format: JPG (oder WebP)
- Mindest-Breite: 800 px
- Empfohlene Größe: 1200 × 1200 px (quadratisch oder querformat 4:3)
- Dateigröße: max. ca. 200 KB pro Bild (komprimieren!)
- Farbprofil: sRGB

Solange Bilder fehlen, zeigt die Site einen neutralen Platzhalter mit Künstlername. Sobald die Datei vorhanden ist, wird sie automatisch eingebunden.
